tinyb
=====

A simple brainfuck interpreter written in C.

Examples:

Hello world:
```
++++++++++[>+++++++>++++++++++>+++>+<<<<-]>++.>+.+++++++..+++.>++.<<+++++++++++++++.>.+++.------.--------.>+.>.
```
